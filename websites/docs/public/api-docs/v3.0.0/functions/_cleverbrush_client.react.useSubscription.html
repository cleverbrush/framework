    Function useSubscription

    • useSubscription<TOutgoing, TIncoming = never>(
          subscribe: () => Subscription<TOutgoing, TIncoming>,
          options?: UseSubscriptionOptions,
      ): UseSubscriptionState<TOutgoing, TIncoming>

      React hook that connects to a WebSocket subscription and provides reactive state.

      Type Parameters

      • TOutgoing
      • TIncoming = never

      Parameters

      • subscribe: () => Subscription<TOutgoing, TIncoming>

        A function that creates a Subscription handle (e.g. () => client.live.events()).

      • options: UseSubscriptionOptions = {}

        Hook options (enabled, maxEvents).

      Returns UseSubscriptionState<TOutgoing, TIncoming>

      Reactive subscription state: lastEvent, events, state, send, close.

      Example

      function LiveFeed() {
          const { events, state, send } = useSubscription(
              () => client.live.events(),
              { maxEvents: 100 }
          );

          return (
              <div>
                  <p>Status: {state}</p>
                  {events.map((e, i) => <div key={i}>{e.message}</div>)}
                  <button onClick={() => send({ text: 'hello' })}>Send</button>
              </div>
          );
      }
